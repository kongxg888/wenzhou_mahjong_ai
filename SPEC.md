## 二、财神系统

### 翻财神
1. 牌墙最后一张翻开
2. 翻开牌的**下一张**为财神
3. 例：翻出"三万"(3) → 财神是"3万"(3)

### 财神规则
| 特性 | 说明 |
|------|------|
| 万能性 | 可代替任意牌凑顺子、刻子、对子 |
| 白板 | 固定代替财神牌（id=33） |
| 禁打 | 不能打出 |
| 禁操作 | 不能吃、碰、杠 |

### 财神ID计算
```python
def get_caishen_id(flip_card_id):
    if flip_card_id == 33:  # 白板
        return 33
    return (flip_card_id + 1) % 34
```

## 三、胡牌牌型

### 3.1 基本牌型公式
```
AA + ABC×5 = 17张

AA  = 1对将（2张）
ABC = 顺子（3张）×5
```

### 3.2 基本牌型变种

| 类型 | 公式 | 说明 |
|------|------|------|
| 标准型 | AA + ABC×5 | 1对将 + 5顺子 |
| 带刻型1 | AA + ABC×4 + AAA | 1对将 + 4顺子 + 1刻子 |
| 带刻型2 | AA + ABC×3 + AAA×2 | 1对将 + 3顺子 + 2刻子 |
| 带刻型3 | AA + ABC×2 + AAA×3 | 1对将 + 2顺子 + 3刻子 |
| 带刻型4 | AA + ABC + AAA×4 | 1对将 + 1顺子 + 4刻子 |

### 3.3 8对牌型

| 类型 | 条件 | 有无财神 |
|------|------|----------|
| 硬8对 | AA×8 + 任意1张 | 无财神 |
| 软8对 | AA×7 + BB + 任意1张 | 有财神 |

### 3.4 特殊牌型

| 类型 | 条件 | 番数 |
|------|------|------|
| 3财神强制胡 | 手持3财神但不符合任何牌型 | ×2 |
| 特殊4翻 | 3财神归位 + 硬8对7对+3财神 | ×8 |

## 四、胡牌等级（核心）

### 4.1 四级制度

```
        ┌─────────────────────────────────────┐
        │            胡牌等级                  │
        ├──────┬────────────────────────────┤
        │ 软牌 │ 倍数×1                      │
        │ 硬牌 │ 倍数×2                      │
        │ 双翻 │ 倍数×4                      │
        │ 特殊 │ 倍数×8                      │
        └──────┴────────────────────────────┘
```

### 4.2 软牌 (×1)
胡牌中**用到财神代替其他牌**（非归位）

### 4.3 硬牌 (×2)
满足以下**任一条件**：
- [ ] 自摸胡牌
- [ ] 无财神参与
- [ ] 财神归位（本尊使用）
- [ ] 抢杠胡
- [ ] 软8对
- [ ] 3财神强制胡

### 4.4 双翻 (×4)
**硬牌**基础上再满足：
- [ ] 硬8对（无财神纯8对）
- [ ] 3财神+正常牌型
- [ ] 天胡（庄家第一张即胡）
- [ ] 地胡（闲家未抓牌就胡）
- [ ] 单吊（吃碰杠后单张）
- [ ] 全球神（只剩财神单吊）
- [ ] 碰碰胡（5刻子+1对）
- [ ] 杠上开花
- [ ] 双财神归位
- [ ] 半清（一色+风字）
- [ ] 清一色（全一色）

### 4.5 特殊4翻 (×8)
- 3财神归位 + 硬8对7对+3财神

## 五、连庄规则

| 连庄次数 | 倍数 | 触发 |
|----------|------|------|
| 第1次 | ×1 | 首局/换庄后 |
| 第2次 | ×2 | 庄家连胡 |
| 第3次 | ×3 | 庄家再连 |
| 第4次 | ×4 | 庄家三连 |
| ≥4次后 | 换庄 | - |

### 换庄条件
- 闲家胡牌 → 换庄
- 流局 → 庄家连庄

## 六、计分公式

### 6.1 基础公式
```
总分 = 胡牌倍数 × 连庄倍数 × 底分 × (1 + 财神个数)
```

### 6.2 自摸加倍
```
总分 = 基础分 × 1.5
```

### 6.3 财神加分
```
财神分 = 财神个数 × 胡牌倍数 × 连庄倍数 × 底分
```

### 6.4 杠分
| 类型 | 加分 | 触发条件 |
|------|------|----------|
| 明杠 | +1分 | 对手打出第4张时杠 |
| 加杠 | +1分 | 碰后摸到第4张 |
| 暗杠 | +2分 | 自己摸到4张 |

### 6.5 完整公式
```
对方支付 = (胡牌倍数 × 连庄倍数 × 底分 × (1 + 财神个数)) + 财神分 + 杠分
```

## 七、动作流程规则（核心闭环）

### 7.1 出牌后的处理流程

```
玩家A出牌
    │
    ├── 玩家B吃 ──────────→ 玩家B直接出牌（不摸牌）
    │
    ├── 玩家B碰 ──────────→ 玩家B直接出牌（不摸牌）
    │
    ├── 玩家B明杠 ────────→ 补牌 → 检查杠上开花 → 玩家B出牌
    │
    ├── 玩家B加杠 ────────→ 摸牌后继续出牌（不再补牌）
    │
    ├── 玩家B暗杠 ────────→ 补牌 → 检查杠上开花 → 玩家B出牌
    │
    ├── 玩家B过 ──────────→ 摸牌 → 玩家B出牌
    │
    └── 玩家B胡 ──────────→ 终局
```

### 7.2 关键规则

| 规则 | 说明 |
|------|------|
| 吃碰后不摸牌 | 吃完/碰完直接进入出牌阶段，不能摸牌 |
| 明杠/暗杠补牌 | 杠后从牌墙补1张，检查杠上开花 |
| 加杠不补牌 | 本身就是摸牌动作，不重复补牌 |
| 过手 | 过牌后摸1张，然后出牌 |
| 荒庄 | 牌墙≤8张时禁止吃/碰/杠 |

### 7.3 最后八张规则
牌墙剩余≤8张时：
- 终结牌局
- 流局

## 八、速算口诀

```
第一步：判断软/硬/双翻 → 确定倍数
第二步：数手中财神个数
第三步：看连庄次数
第四步：套公式
```

### 示例

| 场景 | 倍数 | 连庄 | 财神 | 底分=2 | 结果 |
|------|------|------|------|---------|------|
| 软牌1财神首连 | ×1 | ×1 | 1 | 2 | 1×1×2×2=4 |
| 硬牌自摸0财神二连 | ×2 | ×2 | 0 | 2 | 2×2×2×1=8 |
| 双翻清一色2财神三连 | ×4 | ×3 | 2 | 2 | 4×3×2×3=72 |
| 双翻3财神4连 | ×4 | ×4 | 3 | 2 | 4×4×2×4=128 |
